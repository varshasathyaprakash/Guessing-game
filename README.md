# Guessing Game

A simple guessing game where the user tries to guess a randomly generated number within a specified range. The game provides hints if the guess is too high or too low, and the user can quit the game at any time.

## Key Features
- User can define the maximum number for the guessing range.
- Provides feedback if the guess is too high or too low.
- The user can quit the game by typing "quit".
- A message is displayed if the user correctly guesses the number.

## Technologies Used
- **HTML**: Structure of the webpage.
- **CSS**: Styling the interface.
- **JavaScript**: Logic and functionality of the game.

## How to Play
1. Enter the maximum number for the guessing range.
2. Make a guess.
3. Follow the hints to adjust your guess or type "quit" to exit.
